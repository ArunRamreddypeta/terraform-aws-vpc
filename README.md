# AWS VPC Creation using Terraform

## 📌 Project Overview
This project demonstrates how to provision a basic AWS VPC and a public subnet using Terraform with a modular approach.

## 🧱 Architecture
- 1 VPC
- 1 Public Subnet

## 🛠️ Tools Used
- Terraform
- AWS

## 📂 Project Structure
- Root module calls a reusable VPC module
- Modular design for scalability

## 🚀 Steps to Run

```bash
terraform init
terraform plan
terraform apply
